Configure insight prompts

        

          Every card on /my-insights is driven by the prompt template + data scope you set here.
          When a user clicks Refresh on a card, the worker gathers their data (the slices
          you tick below), substitutes it into {context}, sends the whole thing to the engine,
          and writes the response back as markdown. Changes take effect on the next run.
